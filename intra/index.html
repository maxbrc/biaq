Funktionen

Die Verbindung über das VPN zu unserem Server ist besser verschlüsselt als eine einfache HTTPS Verbindung über das Internet.
Bei aktiver VPN Verbindung müsste außerdem die Anfrage von deinem Gerät erst wieder "ins Internet" und wieder an unseren Server gehen,
was ziemlich überflüssig ist, wenn man schon einmal mit dem Server per VPN verbunden ist.


Übersetzt: Verbindung mit VPN = Mit Intranet verbunden

Wie kann ich es benutzen?

Es läuft stets nach folgendem Muster:
<service>.intra.biaq.de

Ersetze hierbei natürlich <service> nach Bedarf (z. B. nextcloud.intra.biaq.de)
